            Nice article!

            Perhaps worthwhile to mention (obvious to many, but perhaps not all readers) that when a class method is invoked through the object instance, you get the class passed that belongs to the specific object (i.e. the type of that object), not necessarily the class in which the class method was defined.

            An example:

            class Animal:
                @classmethod
                def showcls(cls):
                    print("The class is {cls!r}".format(cls=cls))


            class Dog(Animal):
                pass


            spaniel = Dog()
            spaniel.showcls()
            The class is <class 'Dog'>
